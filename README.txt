### versions used ###
1. JAVA version 1.8.0_71
2. Maven version 
3. Eclipse Neon.2 (4.6.2)
4. testng version 6.8

### How to run ###
Unzip the package in any desired directory example C://GOJEK OR /Users/username/Documents/GOJEK. GO inside ParkingLot directory created after unzip. I have provided a script for running the tool for command as well as interactive mode

# for interactive mode type 
./parking_lot 
and hit enter, start giving commands on standard input

# for file mode type 
./parking_lot <input_file_name>
Example: ./parking_lot file_inputs.txt

### How to build ###
run mvn clean install from git prompt or from eclipse
